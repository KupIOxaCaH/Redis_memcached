### Задание 1. Кеширование 

Приведите примеры проблем, которые может решить кеширование. 

*Приведите ответ в свободной форме.*

---

#### Решение 1

Кеширование позволяет устранить следующие ключевые проблемы:

Высокая задержка: Обращение к диску или удаленной базе данных занимает много времени. Кеш хранит данные в оперативной памяти, что позволяет отдавать их пользователю практически мгновенно.

Высокая нагрузка на базу данных: При большом количестве одновременных запросов база данных может не справляться (тормозить или падать). Кеш берет на себя часть запросов (особенно на чтение), снижая нагрузку на основной источник данных.

Избыточные вычисления: Если система постоянно вычисляет одни и те же «тяжелые» отчеты, это тратит ресурсы процессора впустую. Кеширование готового результата позволяет выполнить вычисление один раз и отдавать его многократно.
​
Стоимость инфраструктуры: Масштабирование мощных баз данных стоит дорого. Внедрение более дешевых кеширующих серверов позволяет обслуживать больше пользователей без покупки дорогих лицензий и «железа» для БД.
